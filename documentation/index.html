Getting started

Welcome

Hello there! Thanks for buying Garth. This is a Gardening Bootstrap template with solid design especially crafted to help you built for Gardening websites with huge variety of options.

This Template is suitable for all type of gardening related website. This template built using Bootstrap v5.0.0 Framework, Google Fonts, Font-awesome v5.15.3, . It is highly recommended that you are familiar in a way with at least HTML5 / CSS3 / Basic Js.

So let's jump directly to it, and start everything from scratch.

Downloading Garth

Once you download the zip file, unzip it. You should have a folder with the following folder & files by now.

Folder Contents
Documentation Documentation and help instructions.
Templates css, fonts, images, js, scss

Installation

Install the template via ftp :

  1. Connect to your website host, using some of many FTP clients, for example FileZilla.
  2. Go to your default server folder, and copy Garth into that folder.

Setting your fonts

To use google fonts, head over to Google's Fonts and add whichever font you want to your collection. Once you finished, follow the instructions and place the generated link in your HTML's file <head> tag.
The link should look like this : 


<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@200;300;400;600;700;800&display=swap" rel="stylesheet">
            
 
body {
  font-family: "Poppins", sans-serif;
  
}

General layout

Every page is built with the latest standards of HTML5, validated with the W3C HTML Validation tool and it is compatible with the latest Bootstrap v5.0.0 Framework.

All pages have 3 main parts:

  1. Navbar
  2. Sections with content
  3. Footer
